### PlaySequence ###

===========================================================================
DESCRIPTION:

The PlaySequence project provides an example of simple MIDI file playback using the MusicSequence API. The file is read in and played back, with options for specifying track number, start/stop times, and output file for render

===========================================================================
BUILD REQUIREMENTS:

Mac OS X v10.7 or later
Xcode v4.3 or later

===========================================================================
RUNTIME REQUIREMENTS:

Mac OS X v10.7 or later

===========================================================================
PACKAGING LIST:

- Main file for project
